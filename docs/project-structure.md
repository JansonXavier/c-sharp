# Project Structure

## Hello World Project

Located in `hello-world/` folder.

### `Program.cs`

Main entry point. Contains:

- `Main()` method - program entry point
- `Console.WriteLine()` - output to console

### `HelloWorld.csproj`

Project configuration file:

- `OutputType` - executable type
- `TargetFramework` - .NET version (net9.0)
- `RootNamespace` - default namespace

## Key Concepts

- **Namespace** - organizes code (`HelloWorld`)
- **Class** - container for code (`Program`)
- **Main method** - runs on program start
- **Console** - handles input/output
